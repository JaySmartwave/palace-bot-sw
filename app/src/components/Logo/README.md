## Logo Component
A component that is a logo image.

### Example

```js
<Logo
  inverse
/>
```

### Props

| Prop          | Type     | Default     | Possible Values
| ------------- | -------- | ----------- | ---------------------------------------------
| **inverse**    | Boolean   |             | True or false
